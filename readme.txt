Instructions to Run 
* Make sure Scikit-learn from https://scikit-learn.org/stable/install.html is installed. 
* Unzip the zip file and open the folder in jupyter. 
* Open sa.ipynb file and run. 
* You should see the different phases of running I.e. preprocessing, training and done when file is running. 
* Once done, predictedScores.dat will be generated in the same folder which is our predicted reviews file.
